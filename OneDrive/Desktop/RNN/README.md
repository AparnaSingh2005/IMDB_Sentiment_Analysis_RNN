#  IMDB Sentiment Analysis using Recurrent Neural Networks (RNN)

A Deep Learning project that performs **binary sentiment classification** on IMDB movie reviews using a **Recurrent Neural Network (RNN)** architecture.



## Problem Statement

Given a movie review, classify whether the sentiment expressed is:

-  Positive  
-  Negative  

This project demonstrates practical implementation of:
- Text preprocessing
- Word embeddings
- Sequence modeling using RNN
- Model evaluation and prediction

##  Dataset

- **Dataset**: IMDB Movie Review Dataset  
- **Total Reviews**: 50,000  
  - 25,000 Training  
  - 25,000 Testing  
- Balanced binary classification dataset
